# KosmoLingua (Kuznetsov Language)

**The World’s First Universal Emoji Meta-Language**  
Built on the standard RGI emoji set that everyone already knows.

---

## 🚀 Highlights

- **Universal & Familiar**: Only standard emojis – no new icons to learn.  
- **Minimal Core**: ~300 emojis covering all basic parts of speech, emotions, relations and modifiers.  
- **Instant Speech**: CVC syllable mapping for each emoji, ready for easy TTS.  
- **Simple Grammar**: SVO order; emoji markers for tense, number, questions, negation.  
- **Open Source**: Transparent research, active community, cross-cultural testing.
